# CDSA Notes

These are my personal notes and revision materials for the HTB CDSA exam. There are markdown and PDF copies of each file and one file per module in the SOC analyst pathway. Note that some of the material in these notes is paraphrased from the module, some module material is excluded and some portions of notes are verbatim module material so A) Credit due to the HTB team for coming up with the material in the first place and B) these notes almost certainly will not help anyone revise for the exam.

Reading other peoples notes will not help you for any exam, in particular a practical one like this - I only include these files to keep them in a central location for my own future use.
